# Security Policy

## 支持的版本

| 版本 | 支持状态 |
|------|----------|
| 1.x  | ✅ 当前支持 |

## 报告漏洞

如果你发现了安全漏洞，请通过以下方式私下报告：

1. **GitHub Security Advisories**: [创建私有安全建议](https://github.com/wuhuairline0727/fip-cli/security/advisories/new)
2. 或发送邮件至仓库维护者

请不要在公开的 Issue 或讨论中披露漏洞细节。

## 响应时间

- **确认**: 72 小时内
- **修复评估**: 7 天内
- **修复发布**: 根据严重程度，通常在 30 天内

## 安全最佳实践

使用本工具时请注意：

- **不要在代码中硬编码密码或 API Key** — 使用 `fip-cli config` 管理配置
- **配置文件权限** — `~/.fiprc.json` 应设置为仅当前用户可读（`chmod 600`）
- **浏览器安全** — 本工具通过 WebBridge 控制浏览器，请确保在可信环境中使用
- **敏感数据** — 提取的单据数据可能包含财务信息，请妥善保管输出文件
